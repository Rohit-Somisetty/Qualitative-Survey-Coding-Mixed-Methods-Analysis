# Demo bundle

This folder contains ready-to-share artifacts generated from the synthetic qualitative pipeline.

## Files
- qual_factsheet_latest.html: one-page mixed-methods fact sheet (figures referenced in ./figures).
- qualitative_brief.md: narrative brief with key themes, co-occurrences, and mixed-methods highlights.
- reliability_by_theme.csv: percent agreement and Cohen's kappa per theme.
- figures/: PNG assets embedded in the fact sheet.

## Regenerate
Run `python scripts/export_demo_assets.py` (or call export_demo_assets()) to rebuild with fresh synthetic data.
